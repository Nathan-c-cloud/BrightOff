# Business Model — BrightOff

Date : 2026-03-21

---

## Modèle : Freemium

Le gratuit donne suffisamment de valeur pour accrocher l'utilisateur, mais crée une frustration ciblée qui pousse vers
le premium. Le mécanisme utilisé est le **pattern "teaser"** : montrer qu'une information existe mais la flouter pour
les utilisateurs non premium.

---

## Version Gratuite

Objectif : acquisition et rétention.

- Upload de CV illimité
- Profil complet (extraction IA + modification manuelle)
- 5 offres matchées par jour, avec scores visibles
- Gap Analysis en mode teaser : 1 compétence manquante visible en clair, le reste flouté avec le message "Débloquez X
  autres compétences manquantes avec Premium"
- Suggestions CV en mode teaser : 1 tip visible en clair, le reste flouté
- Tracking de candidatures complet
- Notifications email (nouvelles offres)
- Notifications in-app (nouvelles offres)

---

## Version Premium

**Tarif mensuel : 9,99 €/mois**
**Tarif annuel : 7,99 €/mois** (facturé annuellement)
**Essai gratuit : 7 jours de Premium à l'inscription**

Fonctionnalités débloquées :

- Offres matchées illimitées
- Gap Analysis complet : toutes les compétences manquantes, impact chiffré, recommandations de formations
- Suggestions CV complètes
- Dashboard de progression des compétences dans le temps
- Plan d'action personnalisé (vision long terme — planning d'apprentissage)

---

## Paiement

- Moyens acceptés : carte bancaire et PayPal
- Prestataire : Stripe (gère les deux moyens de paiement)
